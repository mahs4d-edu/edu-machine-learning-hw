## Data

## Setup and Running

first install pipenv in your system:

pip3 install pipenv

then navigate to project directory and run:

pipenv install

now you can activate the virtual environment with following command:

pipenv shell

then navigate to src folder and run the main.py file:

cd src/

python nb.py
